📸 Keep Your Moments

Keep Your Moments is a simple photo memory web application built with Django that allows users to store and organize photos of their special life events such as graduations, weddings, birthdays, and travels.

The goal of this project is to practice Django backend development, file uploads, authentication, and deployment while building a real-world application.

🌟 Features

     👤 User registration and login

    📁 Create and manage photo albums

    🖼️ Upload photos with captions

    🗂️ Organize memories by events

    🧑‍💻 Django admin panel for management

Users can store memories such as:

    🎓 Graduation

    💍 Wedding

    🎂 Birthday

    ✈️ Trips

    👨‍👩‍👧 Family events
